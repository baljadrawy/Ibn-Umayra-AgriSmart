# 🌿 بوت تقويم ابن عميرة الزراعي

بوت تلقرام مبني بـ Python يقدم التقويم الزراعي التاريخي لابن عميرة معدَّلاً حسب المنطقة الجغرافية.

## الأوامر المتاحة

| الأمر | الوظيفة |
|-------|---------|
| `/start` | البداية واختيار المنطقة |
| `/today` | النجم الحالي وتوصيات اليوم |
| `/weather` | الطقس الحي ومقارنته بالتقويم |
| `/crop` | المحاصيل والأنشطة الموصى بها |
| `/calendar` | أنجم الشهر الحالي |
| `/setlocation` | تغيير المدينة أو المنطقة |
| `/subscribe` | اشتراك في التنبيهات الصباحية |
| `/unsubscribe` | إلغاء التنبيهات |

## التثبيت والتشغيل

### 1. احصل على توكن البوت
- افتح تلقرام وابحث عن [@BotFather](https://t.me/BotFather)
- أرسل `/newbot` واتبع التعليمات
- انسخ الـ Token

### 2. ثبّت المتطلبات

```bash
cd telegram-bot
pip install -r requirements.txt
```

### 3. شغّل البوت

```bash
export TELEGRAM_BOT_TOKEN="your_token_here"
python bot.py
```

### تشغيل في الخلفية (Linux/Mac)

```bash
nohup python bot.py &> bot.log &
```
