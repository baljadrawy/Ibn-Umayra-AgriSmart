# التقويم الزراعي المطور (ابن عميرة) - Ibn Umayra AgriSmart

نظام زراعي ذكي يدمج الخبرة التراثية السعودية لتقويم ابن عميرة الزراعي مع تقنيات الذكاء الاصطناعي الحديثة وبيانات الطقس الحية.

## الرؤية
يهدف هذا المشروع إلى تحويل تقويم ابن عميرة من "صورة ثابتة" إلى نظام رقمي تفاعلي يساعد المزارعين في المملكة العربية السعودية على اتخاذ قرارات زراعية أدق بناءً على الأنواء والطقس الواقعي.

## المميزات الرئيسية
- **التقويم الزراعي الدقيق:** قاعدة بيانات مهيكلة لكامل تقويم ابن عميرة لعام 2026.
- **مقارنة الطقس الحية:** ربط آلي مع Open-Meteo API لمقارنة الواقع الميداني مع التقويم التاريخي.
- **المستشار الزراعي الذكي:** وكيل ذكي مدعوم بـ AI للإجابة على أسئلة المزارعين بلغة طبيعية.
- **دليل المحاصيل:** معلومات مفصلة عن المحاصيل الصحية والمناسبة لمناخ المملكة.
- **واجهة مستخدم عصرية:** تصميم مستوحى من بساطة وأناقة Apple مع دعم كامل للغة العربية (RTL).

## التقنيات المستخدمة
- **الواجهة الأمامية:** Next.js 15 (App Router), React 19.
- **التنسيق:** Tailwind CSS, Shadcn UI.
- **الذكاء الاصطناعي:** Genkit (Google Gemini 2.5).
- **قاعدة البيانات:** Firebase Firestore.
- **الطقس:** Open-Meteo API.

## كيفية البدء
1. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
2. قم بتشغيل المشروع في بيئة التطوير:
   ```bash
   npm run dev
   ```
