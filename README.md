# التقويم الزراعي المطور (ابن عميرة) - Ibn Umayra AgriSmart

نظام زراعي ذكي يدمج الخبرة التراثية السعودية لتقويم ابن عميرة الزراعي مع تقنيات الذكاء الاصطناعي الحديثة وبيانات الطقس الحية.

## الرؤية
يهدف هذا المشروع إلى تحويل تقويم ابن عميرة من "صورة ثابتة" إلى نظام رقمي تفاعلي يساعد المزارعين في المملكة العربية السعودية على اتخاذ قرارات زراعية أدق بناءً على الأنواء والطقس الواقعي، مع مراعاة الاختلافات المناخية بين مناطق المملكة.

## المصادر المعتمدة
يعتمد النظام على مصادر بيانات دقيقة وموثوقة:
1. **تقويم ابن عميرة (2026):** المرجع الأساسي للأنواء والمواسم الزراعية.
2. **Open-Meteo API:** لجلب بيانات الطقس الحية واللحظية لكافة مدن المملكة.
3. **تصنيف الأحزمة المناخية:** نظام ذكي يعتمد على بيانات المركز الوطني للأرصاد لتعديل التواقيت حسب المنطقة.
4. **بيانات MEWA:** قاعدة بيانات المحاصيل والزراعة في المملكة العربية السعودية.

## المميزات الرئيسية
- **تحديد الموقع الذكي:** التعرف الآلي على موقع المزارع وتقديم نصائح مخصصة لحزامه المناخي.
- **مقارنة الطقس الحية:** ربط آلي لمقارنة الواقع الميداني مع التقويم التاريخي (Match Score).
- **المستشار الزراعي الذكي:** وكيل ذكي مدعوم بـ AI (Gemini) للإجابة على أسئلة المزارعين.
- **دليل المحاصيل الصحية:** معلومات مفصلة وصور عالية الجودة للمحاصيل المناسبة للمملكة.

## التقنيات المستخدمة
- **الواجهة الأمامية:** Next.js 15 (App Router), React 19.
- **التنسيق:** Tailwind CSS, Shadcn UI.
- **الذكاء الاصطناعي:** Genkit (Google Gemini 2.5).
- **قاعدة البيانات:** Firebase Firestore.
- **الطقس:** Open-Meteo API.
