# التقويم الزراعي المطور (ابن عميرة) - AgriSmart

نظام زراعي ذكي يدمج الخبرة التراثية لـ **تقويم ابن عميرة** مع تقنيات الذكاء الاصطناعي وبيانات الطقس الحية لخدمة المزارع السعودي.

## 🌟 الرؤية
تحويل التقويم التراثي من مجرد "معلومات ثابتة" إلى نظام تفاعلي يقدم نصيحة مخصصة لكل مزارع بناءً على موقعه الدقيق وظروف الطقس اللحظية.

## 🚀 المميزات الرئيسية
- **تحديد الموقع الذكي:** التعرف الآلي على المدينة وتصنيفها ضمن الأحزمة المناخية الخمسة للمملكة.
- **التعديل الزمني الآلي:** حساب فارق الأيام (تقديم/تأخير) عن التقويم الأصلي لكل مدينة ومحافظة.
- **مقارنة الطقس الحية:** ربط مع Open-Meteo لحساب "نسبة التوافق" بين الواقع والتقويم التراثي.
- **المستشار الذكي (AI):** وكيل مدعوم بـ Gemini 2.5 للإجابة على أسئلة المزارعين بدقة مكانية وزمانية.
- **دليل المحاصيل الصحية:** معلومات وصور عالية الجودة للمحاصيل المتوافقة مع مناخ المملكة.

## 🛠️ التقنيات المستخدمة
- **Frontend:** Next.js 15, React 19, Tailwind CSS.
- **Backend/AI:** Firebase Firestore, Genkit, Gemini 2.5 Flash.
- **APIs:** Open-Meteo API.
