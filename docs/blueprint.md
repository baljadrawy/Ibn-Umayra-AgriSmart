# **App Name**: Ibn Umayra AgriSmart

## Core Features:

- Calendar Data Management: Structure the Ibn Umayra agricultural calendar data into a searchable and accessible PostgreSQL database, capturing annual forecasts for climate and agricultural advice.
- Climate Zone Localizer: Dynamically adapt traditional agricultural recommendations based on the user's selected climate zone and specific local adjustments, including crop suitability and timing shifts.
- Personalized Proactive Alerts: Deliver proactive, personalized agricultural advice and warnings via Telegram, tailored to the user's climate zone, current calendar period, and crop preferences.
- Live Weather Integration & Comparison: Integrate real-time and historical weather data from external APIs (e.g., Open-Meteo) to compare against the calendar's predictions, offering a 'match score' and refined advice.
- AI-Powered Advisory Tool: Provide a smart AI tool that leverages LLM technology (e.g., Ollama) and an enriched knowledge base to answer farmers' free-form questions in their native language, offering context-aware agricultural insights.
- Interactive Web Calendar & Dashboard: Offer an intuitive web interface for a visual annual calendar, daily and upcoming recommendations, and a crop guide, all localized to the user's climate zone.
- Farm Activity Logging: Enable users to log their farming activities such as planting, harvesting, or observations, associating them with specific crops and calendar dates for historical tracking.

## Style Guidelines:

- Color scheme: Light. Primary color: A grounded, natural green (#339944) reflecting agriculture and vitality. Background color: A very light, desaturated green (#EEF8EF) to provide a fresh, clean canvas. Accent color: A vibrant yellow-green (#D5ED5E) for call-to-actions, alerts, and highlighting key information, evoking growth and energy.
- Headline font: 'PT Sans' (humanist sans-serif) for its modern yet approachable feel. Body font: 'Inter' (grotesque-style sans-serif) for excellent readability and clarity, particularly crucial for detailed agricultural advice and data presentation.
- Employ a set of clear, illustrative icons for crops, weather conditions, and various agricultural activities. Icons should be modern, easily distinguishable, and designed to support the RTL layout and Arabic text content, enhancing visual communication.
- Implement a right-to-left (RTL) layout globally to naturally accommodate the Arabic interface. Prioritize clean, organized data presentation in dashboards and calendar views. Design responsive layouts to ensure optimal experience across various devices for all target users.
- Incorporate subtle and functional animations for user interactions such as navigation transitions, data loading indicators, and dynamic calendar updates. These animations should enhance the user experience without being distracting or impeding usability.
